> Python 3.14 已到货！

经过多轮测试，Python 3.14 更新已推送至安同 OS 稳定源。根据您选用的镜像源，您可以在 12 小时以内使用如下命令安装相关更新：

```
oma upgrade
```

如果您在更新 Python 3.14 时遇到问题或有任何建议，欢迎您来我社[各聊天群组](https://aosc.io/contact)与我们联系反馈。
