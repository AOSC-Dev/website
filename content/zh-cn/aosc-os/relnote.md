---
title: 发行注记：2026 年 3 月
---

本次发布的安同 OS 系统安装介质包含至 2026 年 3 月 12 日为止的各类系统组件更新及安全修复。除常规更新外，2026 年 3 月发行更新主要包含将安同 OS 核心包升级至第 13 版 (Core 13)，其中包括 glibc 2.42、GCC 15 等最新的核心运行时及工具链。此外，本版将 Python 更新到了 3.14，并针对龙架构及龙芯三号 (MIPS) 引入了对相应 Debian 架构软件包的安装兼容支持（即 `loongarch64` 映射为 `loong64`，`loongson3` 映射为 `mips64el`）。

本版安装盘及发行方式亦有较大改动：安装盘采用 EROFS 文件系统封装各系统环境镜像，在同等内容下，安装盘尺寸缩减约 5%，安装耗时缩短近 16%。安装盘内容方面，本版镜像在原有 NVIDIA 驱动的基础上，引入了基于 NVIDIA 开源 Linux GPU 内核模块 (NVIDIA Linux Open GPU Kernel Module) 的驱动，可更好地支持 GeForce GTX 16 及 RTX 系列等较新的 NVIDIA 显卡（同时保留了 NVIDIA 580 版显卡驱动，以继续支持 GTX 10 系列及更老的显卡型号）；此外，还集成了龙芯 LoongGPU、格兰菲及兆芯的集成显卡驱动。综合而言，本版安装盘可支持更多不同种类的设备，提供更便捷的安装体验。

**同时，考虑到使用频次低且测试覆盖不足，自本次发行起，我们将不再提供 LiveKit 恢复盘，一律以安装盘为准。此外，为了节省服务器空间，我们还取消了 `desktop+nvidia`（预装 NVIDIA 的桌面版）系统压缩包发行。**

硬件支持方面，本版通过更新 Linux 内核、设备固件等，增强了对新老设备的兼容性。2026 年 3 月发行更新正式支持了联想 ThinkBook 14 G8+ IPH 等基于 Intel Panther Lake 平台的笔记本电脑，加入了龙芯 2K3000/3B6000M SoC 集成的 LoongGPU LG200 的显卡驱动，并初步修缮了龙芯 7A1000（龙架构使用 LoongGPU 驱动，龙芯三号 (MIPS) 使用 Linux 主线内核的 `loongson` DRM 驱动）集成显卡的支持。

**请注意：如果您已安装安同 OS，本发行注记提到的修复及变化均已通过系统更新推送，您无需重装系统或额外安装任何软件包。**

平台支持
---

本次更新包含大量针对硬件平台支持的更新和修复，主要更改如下：

### 硬件支持更新

- 更新 Linux 内核至 6.18.16，带来了新平台支持、各类修复及优化，并向后移植了裕太微 YT6801 以太网卡的主线化内核驱动，改善性能及稳定性（平台特有的支持更新及改进请见下文）
- 更新 NVIDIA 驱动至 590.48.01 并引入 NVIDIA 580 版驱动 (`nvidia+580`) 以继续支持 GeForce GTX 900 及 10 等系列显卡
- 更新格兰菲 Arise 1000/2000 系列显卡驱动至 25.00.51（龙架构版本暂时滞后于 25.00.46）

### 设备支持 — x86-64

- 更新 Linux 设备固件、SOF 音频设备固件等到当前最新版，修复 AMD Phoenix APU 硬件平台偶发桌面无响应或锁死的问题，并新增 Intel Panther Lake 平台音频、图形、摄像头 (Intel Image Processing Unit 7)、NPU 及网络设备的支持
- 更新兆芯 C-960（内置于兆芯开先 KX-6000/6900）集成显卡驱动至 21.00.86
- 更新兆芯 C-1080/1190（内置于兆芯开先 KX-6000G/7000）集成显卡驱动至 26.00.45
- 修复兆芯 C-4600 设备启动时内核崩溃的问题
- 新增针对兆芯、海光等平台设备的 GRUB 引导器规避，避免引导器遗失等平台专属问题

### 设备支持 — 龙架构

- 更新 LoongGPU 显示驱动 1.0.2（详细版本：1.0.2+0ky11.1~0.10），加入了龙芯 2K3000/3B6000M SoC 集成的 LoongGPU LG200 的显卡驱动，并启用初步 Wayland 支持，初步修复了龙芯 7A1000 桥片集成显卡的显示支持
- 更新 Memtest86+ 至 8.00，新增龙芯 2K3000/3B6000M 平台支持，并修复 L1/L2/L3 带宽计算不准确的问题
- 修复龙芯 3B6000 处理器拓扑报告错误
- 修复龙架构平台使用 AMD RDNA 3 显卡时偶发驱动崩溃的问题
- 改善龙芯 2K3000/3B6000M GPIO 及 Multi-Channel DMA 总线支持

### 设备支持 — AArch64

- 新增此芯科技 P1 (SKY1) SoC 的初始支持
- 新增针对部分华为擎云 W510 机型的 GRUB 引导器规避，避免系统启动后不定时断电或重启的问题

### 设备支持 — 龙芯三号 (MIPS)

- 修复龙芯三号 (MIPS) 设备上的龙芯 7A GMAC（以太网接口）及笔记本内置键盘支持
- 初步修缮了龙芯 7A1000 桥片集成显卡的显示支持（仅限台式机等无显示控制器桥的平台，笔记本暂无开箱支持）

### 设备支持 — RISC-V（64 位）

- 初步加入平头哥 TH1520 SoC 的支持
- 改善算能 SG2042/2044 及 Milk-V Pioneer 整机的支持，提高 PCIe 外设兼容性

关键组件更新
---

本次系统发行更新包含许多组件更新，其中较为关键的更新及修复有：

- 小熊猫包管理 (oma) 更新至 1.24，新增镜像源延迟测试工具、提权回落机制等新功能，并改善了与 APT 元数据获取及密码学策略配置的兼容性
- 更换默认音频系统为 PipeWire（原为 PulseAudio），可更好地支持各类复杂音频设备配置（如 > 2 个扬声器的笔记本电脑）
- 默认启用 NTSYNC 内核模块，改善部分 Windows 应用程序（尤其是游戏）的性能
- 针对龙架构及龙芯三号 (MIPS) 引入了使用 Debian 架构名 (`loongarch64` → `loong64`, `loongson3` → `mips64el`) 的软件包的安装支持
- 更新 Python 至 3.14，包含自 Python 3.10 起的各类新语言及运行时特性，同时清理了一大批 Python 软件包的依赖并进一步剥离了 Python 2 依赖
- 更新 Firefox 及 Thunderbird 至 148.0，修复了多个严重及高危漏洞
- 更新 LibTIFF 至 4.7.1，修复自 2023 年以来在 LibTIFF 中披露的 29 处安全漏洞（含 8 个高危漏洞），并通过 `libtiff-5` 提供老旧软件的兼容性
- 更新 BIND、curl、Exiv2、FreeRDP、FreeType、GLib、glibc、GNU InetUtils、GnuPG、GnuTLS、ImageMagick、OpenSSL、PyCA Cryptography 等其他系统组件，共修复了其中发现的数十个严重或高危安全漏洞
- 更新 Vulkan 图形及计算栈至 1.4.335.0，支持更多新增扩展
- 更新 GNOME 桌面组件及运行时至 49，结束三年以来绑定 GNOME 42 的历史
- 更新时区数据库 (`tzdata`) 至 2026a 并换用兼容性更好的 "fat" 格式配置数据，修复部分程序的兼容性问题
- 更新 SDL 运行时到 SDL3，并通过兼容层继续提供 SDL 1.2 及 SDL2，提高与新设备及应用程序的兼容性
- 更新 OpenSSL 1.1.1 至由第三方维护的 1.1.1ze：由于 OpenSSL 官方不再提供针对 v1.1.1 分支的免费安全更新，本更新修复自 2024 年以来在 OpenSSL 1.1.1w 中发现的若干安全漏洞
- 更新安同 OS 软件源数据包至 20260321，更新各镜像源数据，更改香港软件源托管方为 [Nearoute Limited](https://nearoute.com/home) 并新增[华中科技大学开源镜像站](https://mirrors.hust.edu.cn/)

功能修缮
---

除功能更新带来的修复外，我们还针对用户反馈的问题对部分系统功能中存在的问题进行了修复：

- 修复了 NVIDIA 驱动未正确屏蔽 Nova 内核驱动，导致显卡驱动无法正确初始化的问题
- 修复了 Binutils 中工具的目标前缀重复（如 `x86_64-aosc-linux-gnu-` 被错误重复了一次），导致软件包内容冗余的问题
- 修复部分架构下 Linux 内核默认文件系统编码非 UTF-8 的问题

安装体验修缮
---

本次更新还包含针对安装盘功能及安装体验的一系列修缮：

- 换用 EROFS 封装各系统环境镜像，同等内容下安装盘尺寸缩减约 5%，安装耗时缩短近 16%
- 集成 NVIDIA 开源 Linux GPU 内核模块 (NVIDIA Linux Open GPU Kernel Module) 的驱动，可更好地支持 GeForce GTX 16 及 RTX 系列等较新的 NVIDIA 显卡（同时保留了 NVIDIA 580 版显卡驱动，支持 GTX 10 系列及更老的显卡型号）
- 集成了龙芯 LoongGPU、格兰菲及兆芯集成显卡驱动。综合而言，本版安装盘可支持更多不同种类的设备，提供更便捷的安装体验
- 默认禁用串口终端设备探测，降低默认情况下造成启动菜单操作迟缓或启动故障的概率（如需使用串口终端，可在引导时按住 Esc 或 F4 键启用）
- 新增针对华硕 XC-KX700M D4（兆芯开先 KX-7000）及紫光 UNIS L3891 G2（兆芯开先 KX-6640MA）安装后由于 UEFI 固件未正确保存 EFI 变量，导致无法正常引导系统的问题规避
- 新增针对 HO-Z6000G 瘦客户机（兆芯开先 KX-6000G）的串口终端设备探测规避，解决启动菜单操作迟缓的问题
- 新增针对华为 D920S10 主板的串口终端设备探测规避，解决安装盘启动后无显示的问题
- 禁用龙芯三号 (MIPS) 平台下的 DMI 信息探测，规避部分使用 LEFI 固件的龙芯 3A3000 平台引导时系统锁死的问题
- 新增关于安同 OS 与安全启动支持的问题说明，详见[《安同 OS 与安全启动》](https://aosc.io/aosc-os/secure-boot)
- 更换“救援与恢复工作区”默认壁纸为喻昌远投稿的摄影作品《南浦大桥》
- **（2026 年 3 月 26 日修订 1）** 修复启动带格兰菲显卡驱动的安装环境时意外启动至试用环境的问题
- **（2026 年 3 月 26 日修订 1）** 改进 GRUB 配置文件对 u-root 等 Linuxboot 实现的兼容性
