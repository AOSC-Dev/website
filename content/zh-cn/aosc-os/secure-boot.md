---
title: 安同 OS 与安全启动
---

安同 OS 不支持安全启动，在引导安同 OS 及安装盘之前，您需要通过 UEFI 固件设置界面禁用安全启动。

如果您的设备预装了 Windows，或者您在启用了安全启动的情况下安装了 Windows，请注意以下几点：

- 如果您需要使用 Windows 玩搭载内核级防作弊系统的在线游戏（如 BattlEye、EA 防作弊、Easy Anti-Cheat 等），我们推荐您使用虚拟机或另一台受支持的设备安装安同 OS
- 在禁用安全启动前，请检查您 Windows 系统的 BitLocker 硬盘加密状态，并在禁用安全启动前关闭 BitLocker
  - 您需要完全关闭 BitLocker，请勿选择“暂停保护”
  - 大多数情况下，如果禁用了安全启动但没有关闭 BitLocker，Windows 可能无法正常启动（Windows 会询问 BitLocker 恢复密钥）

::h2{id="secure-boot-faq"}
安同 OS 不支持安全启动的原因
::

安同 OS 不支持安全启动的原因比较复杂，但总结起来有以下几点：

- 社区没有法律实体，因此无法向微软公司申请并签发用于签名内核及引导器的证书
- 社区认为签发证书费用高昂且无法合理化

感谢您的理解。
