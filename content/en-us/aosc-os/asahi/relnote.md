title: Release Notes: November, 2025
---

AOSC OS (Apple silicon) is a system release for Apple silicon-based Macs. The main changes for the current release is as follows.

## Component Updates

This release contains system updates released up to November 15, 2025. Notably:

- An out-of-the-box experience (OOBE) wizard is now included to help customise basic configurations such as username, timezone, language, and virtual memory (swap).
- Linux Kernel is updated to 6.17.7, enabling GPU acceleration.
