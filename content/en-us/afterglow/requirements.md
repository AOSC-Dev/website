---
title: System Requirements - Afterglow
---

Afterglow targets older (legacy) devices and, as such, its functionalities are limited. However, the pre-installed applications and features still poses requirements for the processor, graphics cards, storage, and Internet connection on your device. 

Below is a table of system requirements to guarantee good user experience.

::div{id="afterglow-requirements-table"}
| Hardware Component | Requirement(s)                                      |
| ------------------ | --------------------------------------------------- |
| Processor          | Any processor compatible with Afterglow             |
| RAM                | 64MiB                                               |
| Graphics           | Any graphics or framebuffer card, or serial console |
| Storage            | 1GiB (Base) or 3GiB (Desktop)                       |
| Networking         | Internet connectivity                               |
::

## Additional Notes

- For information on processors supported by Afterglow, please see [*Instruction Set Architecture Support Matrix - Afterglow*](/afterglow/isa).
- For graphical desktop, SVGA resolution and colour depth is required (800Ã—600 @ 16-bit/65535 colours).
- Storage requirement takes into account the space that may be required for everyday use after installation. Swap area may take up additional space (especially when additional storage space is available).
